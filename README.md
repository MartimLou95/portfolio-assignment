# Public repo for my personal portfolio assignment

Portfolio page built step by step using HTML, CSS, Git.

Live site: https://martimlou95.github.io/portfolio-assignment/

## Project Notes

- I have made the previous project repo private because I didn't commit step by step and therefore missed an important part of the process.
- Using my previous work as reference, code will be added in small steps so the commit history shows the build process and I practice good habits.
- I will use images from my longstanding live website, martimlou.com. Those and other references will stay local in `.reference/` and are not committed. This way I can practice staging before commits.

## Portfolio Page Features

- Semantic HTML structure
- Responsive project grid
- Local project images with alt text
- Hover states
- Contact links
- Step-by-step Git commit history

## Design Notes

The design is intentionally minimal, inspired by my existing colour grading portfolio. I focused on clear navigation, a restrained colour palette, readable spacing, and a simple project grid.
Throughout the process I requested assistance from Claude to learn about different approaches to shaping image behaviour.

Ultimately I decided on a simple image-led project grid, where each project image acts as the link to my original website. This keeps the interface minimal while still making the portfolio feel visual and easy to navigate.

## Future Considerations

For the next assignment (multi-page web portfolio) I will switch the links to my original website for newly created pages that recreate them.

This way I can chain these assignments into a proper portfolio migration from my live Squarespace site.
